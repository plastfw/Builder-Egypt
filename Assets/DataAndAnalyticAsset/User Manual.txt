Для работы с датой, нужно создать сцену Screensaver (Есть в ассете) Настроить как в примере, либо использовать
уже имеющуюся. Ее нужно загружать самым первым уровнем, а эта сцена уже запускает нужную.
Установите все префабы из ассета на каждом игровом уровне.

// Дата
Укажите название своего проекта в поле _dataKeyName, в скрипте Data, для корректной работы сохранений. 
В игровых уровнях используйте методы GameSceneManager, для перезапуска уровней или запуска следующего.
(!) На мобильных устройствах сохранение при выходе нужно делать с помощью метода OnApplicationPause. 

// Аналитика
После интеграции AppMetrica и GameAnalityc , расскоментируйте все строчки в скриптах AnalyticManager, 
AppMetricaObject, GameAnalyticsObject. Создайте 2 ScriptableObject (Create/GameAssets/AppMetricaObject) и
(Create/GameAssets/GameAnalyticsObject) и установите полученные объекты в AnalyticManager.

Обращаться к аналитике можно через объект AnalyticManager,поставленного до этого на сцене.
Обращаться к дате можно через Data, который также должен быть установлен, в качестве объекта на сцене
Обращаться к сцен менеджеру можно через SceneManager, который должен находиться на объекте в сцене.
